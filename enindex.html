Paper Note: Deep Attention SMOTE Data Augmentation (DA-SMOTE-ED)

Addressing interpolation noise caused by inter-class overlap in imbalanced anomaly detection of gas turbines.

Paper Title: Deep attention SMOTE: Data augmentation with a learnable interpolation factor for imbalanced anomaly detection of gas turbines

Journal: Computers in Industry (Elsevier), Vol. 151, 2023

DOI: 10.1016/j.compind.2023.103972

1. Research Background

In health monitoring of industrial equipment like gas turbines, real-world data faces two significant challenges:

  1. Extreme Data Imbalance: Normal operation data dominates, while anomaly data is extremely scarce. Direct training results in biased models.
  2. Inter-class Overlap: Slight anomalies share similar physical features with normal states, leading to mixed regions in the feature space.

2. Limitations of Traditional Data Augmentation

Traditional SMOTE generates new samples via random interpolation between neighbors. With "inter-class overlap," this randomness easily creates synthetic samples in "danger zones" occupied by normal data, producing mislabeled noise that misleads the classifier.

Fig 1. Data distribution comparison between traditional and proposed hybrid resampling methods.

3. Methodology: DA-SMOTE-ED

The paper proposes a hybrid resampling framework (DA-SMOTE-ED) consisting of three core modules:

3.1 Cluster-based Undersampling Mechanism

The method uses K-Means clustering on normal samples. By keeping cluster centers to represent normal data, it alleviates imbalance while preserving the global distribution, avoiding information loss from random deletion.

3.2 Encoder-Decoder based Feature Space Decoupling

A Transformer-based Encoder-Decoder network with self-attention is introduced:

  • Function: Maps raw data into a highly separable feature space.
  • Triplet-Center loss: Pulls same-class samples together and pushes different classes apart, reducing the risk of samples falling into "danger zones."

Fig 2. Schematic of Triplet-Center loss in the feature space.

3.3 Deep Attention-guided Adaptive Interpolation

Fig 3. Network architecture and core attention module of DA-SMOTE-ED.

The DA-SMOTE module uses an attention network to learn the interpolation factor:

  • Adaptive Interpolation: Automatically guides synthetic samples to stay away from the normal cluster center.
  • Reconstruction: High-quality features are mapped back to the original space via the Decoder.

4. Conclusion

Tests on gas turbine and aero-engine datasets demonstrate:

  • Diagnostic Accuracy: Achieved 91.77% Balanced Accuracy on real datasets, outperforming baseline algorithms.
  • High TPR: Anomaly True Positive Rate reached 89.59%, significantly better than GAN-based models.
